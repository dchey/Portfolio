Kdrama Filter
React App Design and Implementation

Overview

In this project, I created a simple React.js app with filtering and sorting functionality with an aggregator. This means that the app has a list of items that the user can sort and/or filter. It also lets users to aggregate the items for some purpose. In this app, I created a list of kdramas and let users to find and shop their favorite dramas.

Goals and Values

The goal of this app is to create a React.js app with sorting, filtering, and aggregator functionalities with states and components.

The value of this app is to allow users to find and shop their favorite kdramas with filtering and sorting functionalities.

Design and Implementation Details

Some usability and design principles that I considered are simplicity and consistency. For simplicity, I used navigation bar on the top to let users find sorting and filtering functionalities quickly. For consistency, I used same design for all the items in both carts and filtered/sorted list, and used the same colors and hovering functionalities for all the filtering/sorting buttons in the navigation bar.

The app lets users to filter by tv channels and genre, and sort by ratings. The aggregating property is a cart that totals the price of dramas.

Conclusion

This was my first time using Javascript and React, so even though there was a lot to cover, the assignment was overall interesting and learned a lot, especially with adding functionalities in websites and how to structure a React program.
